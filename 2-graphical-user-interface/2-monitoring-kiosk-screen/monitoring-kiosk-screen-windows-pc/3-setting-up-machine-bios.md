
### 1. Auto power up on AC power restore:

>Note: Steps are different from one machine to another.

1.1. Open machine bios settings.

1.2. Go to `power` , `AC Power` , `AC Power On` settings.

1.3. Enable it or select `restore AC status`.

### 2. Make sure CMOS battery is working and time is correct.

> Note: If the battery is empty the settings will be lost if the power was lost or device was unplugged.
