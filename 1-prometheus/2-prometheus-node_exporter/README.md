### **What is Node Exporter?**

> The Node Exporter is a **Prometheus exporter** that runs on the target machine and collects metrics about the system's resources, such as CPU, memory, disk, and network usage. These metrics are exposed in a format that Prometheus can scrape and store for further analysis and visualization.
