> **NOTE**: Tabs are 2 space.

> Here I'm using `one job` for the entire app node exporter metrics configs.

> Naming is like the following `app_name` followed by `_blackbox_exporter_job` -> `app_name_blackbox_exporter_job` for easier filtering later.

## (Option_1) In main prometheus `prometheus.yml`:

> It OK if you just add one or two jobs in `prometheus.yml`.

> Things can get over crowded easily.

> Not recommended for big scale.

```bash
vim /etc/prometheus/prometheus.yml
```

**Under `scrape_configs:` add the following job:**
```php
 - job_name: 'blackbox_exporter_job'
   metrics_path: /probe
   params:
     module: # modules and probes already configered in blackbox's configs.
       - http_2xx  # http probe that looks for a HTTP 2xx response.
   static_configs:
     - targets: # the following targets are examples.
       - https://www.google.com
       - http://localhost:9090
       - https://10.0.0.23
       - 10.0.0..15:443
       - 10.0.0.14:3000
       - https://10.0.0.17:8080
   relabel_configs: # this part is added to any blackbox job.
     - source_labels: [__address__]
       target_label: __param_target
     - source_labels: [__param_target]
       target_label: instance
     - target_label: __address__
       replacement: localhost:9115  # The blackbox exporter's real hostname:port.
```

## (Option_2) In a separate YAML file: 

> **Note**: Don't forget to add the `<PATH>/*.yml` to `prometheus.yml` under `scrape_config_files`.

---

**EXP_1:**  
**Create directory for the app:**
```bash
mkdir -p /etc/prometheus/targets/my_electromics_shop
```

**Create target file:**
```bash
vim /etc/prometheus/targets/my_electromics_shop/blackbox_exporter.yml
```

> **NOTE**: Example includes job renaming so that two different jobs get stored under one job name in prometheus database so that we can query multiple jobs with just one `job_name`.

**Create target file:**
```php
scrape_configs:

  - job_name: "my_electromics_shop_blackbox_exporter_job"
    metrics_path: /probe
    params: # commenting this section will load default(first probe in blackbox configs)
      module: #[default] # modules and probes already configered in blackboxs configs .yml.
#        - tcp_connect  # http probe that looks for a HTTP 200 response.
        - http_2xx
    static_configs:

      - targets:
        - my-electromics-shop-app-01.local # APP 1
        labels:
          app: my_electromics_shop
          ip: 10.0.0.10
      #---------------------------------
      - targets:
        - my-electromics-shop-app-02.local  # APP 2
        labels:
          app: my_electromics_shop
          ip: 10.0.0.11
      #---------------------------------
      - targets:
        - https://my-electromics-shop.com # load balancer
        labels:
          app: my_electromics_shop
          ip: 10.0.0.15
      #---------------------------------
    relabel_configs:
      - source_labels: [__address__]
        target_label: __param_target
      - source_labels: [__param_target]
        target_label: instance
      - target_label: __address__
        replacement: localhost:9115  # The blackbox exporter's real hostname:port.

#------------------------------------------------------------------------------

# Databases have their own exporter but we are just gonna monitor it on TCP level.
# In this example we renamed the database TCP job to the main one so that when we query job_name: "my_electromics_shop_blackbox_exporter_job" we get the apps and the database.

  - job_name: "my_electromics_shop_db_blackbox_exporter_job"
    metrics_path: /probe
    params: 
      module:
        - tcp_connect # TCP probe that looks for a 200 response.
#        - http_2xx # http probe that looks for a HTTP 200 response.
    static_configs:
      #---------------------------------
      - targets:
        - my-electromics-shop-database.local:3306 # database
        labels:
          app: my_electromics_shop
          ip: 10.0.0.12
      #---------------------------------
    relabel_configs:
      - source_labels: [__address__]
        target_label: __param_target
      - source_labels: [__param_target]
        target_label: instance
      - target_label: __address__
        replacement: localhost:9115  # The blackbox exporter's real hostname:port.
      # To show all targets under one job name. 
      - source_labels: ["job"]
        target_label: "job"
        replacement: "my_electromics_shop_blackbox_exporter_job"

```
