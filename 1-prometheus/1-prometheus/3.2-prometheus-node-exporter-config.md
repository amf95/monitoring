> **NOTE**: Tabs are 2 space.

> Here I'm using `one job` for the entire app node exporter metrics configs.

> Naming is like the following `app_name` followed by `_node_exporter_job` -> `app_name_node_exporter_job` for easier filtering later.

---
## (Option_1) In main prometheus `prometheus.yml`:

> It OK if you just add one or two jobs in `prometheus.yml`.

> Things can get over crowded easily.

> Not recommended for big scale.

**Create target file:**
```bash
vim /etc/prometheus/prometheus.yml
```

**Under `scrape_configs:` add the following job:**
```php
  - job_name: "local_node_exporter_job"
    static_configs:
      - targets:
        - localhost:9100
        labels:
          app: monitoring_machine # added lablel
          ip: 10.0.0.25 # added label
```

---
## (Option_2) In a separate YAML file: 

> **Note**: Don't forget to add the path of the target directory to the main `prometheus.yml` under `scrape_config_files`.

**EXP_1:**  Using domain names.

**Create directory for the app:**
```bash
mkdir -p /etc/prometheus/targets/my_electromics_shop
```

**Create target file:**
```bash
vim /etc/prometheus/targets/my_electromics_shop/node_exporter.yml
```

```php
scrape_configs:

  - job_name: "my_electromics_shop_node_exporter_job"
    static_configs:
    
      - targets:
        - my-electromics-shop-app-01.local:9100 # APP 1
        labels:
          app: my_electromics_shop
          ip: 10.0.0.10
      #---------------------------------
      - targets:
        - my-electromics-shop-app-02.local:9100  # APP 2
        labels:
          app: my_electromics_shop
          ip: 10.0.0.11
      #---------------------------------
      - targets:
        - my-electromics-shop-database.local:9100 # database
        labels:
          app: my_electromics_shop
          ip: 10.0.0.12
      #---------------------------------
      - targets:
        - my-electromics-shop.com:9100 # load balancer
        labels:
          app: my_electromics_shop
          ip: 10.0.0.15

```

---

**EXP_2:**  Using direct IP.

**Create directory for the app:**
```bash
mkdir -p /etc/prometheus/targets/my_forum
```

**Create target file:**
```bash
vim /etc/prometheus/targets/my_forum/node_exporter.yml
```

```php
scrape_configs:

  - job_name: "my_forum_node_exporter_job"
    static_configs:
    
      - targets:
        - 10.0.0.20:9100 # APP 1
        labels:
          app: my_forum
          machine: app_01
      #---------------------------------
      - targets:
        - 10.0.0.21:9100 # APP 2
        labels:
          app: my_forum
          machine: app_02
      #---------------------------------
      - targets:
        - 10.0.0.22:9100 # database
        labels:
          app: my_forum
          machine: my_forum_database
      #---------------------------------
      - targets:
        - my-forum.com:9100 # load balancer
        labels:
          app: my_electromics_shop
          ip: 10.0.0.25
          machine: my_forum_load_balancer

```
